## Выполненные задания


1. **Установила Docker, проверила установку и запустила тестовый контейнер:**  
   - Выполнена установка Docker Desktop.  
   - Проведена проверка корректности установки.  
   - Запущен тестовый контейнер (`docker run hello-world`).


2. **Скачала образ Ubuntu:**  
   - Выполнена команда `docker pull ubuntu`.  
   - Запущен интерактивный контейнер: `docker run -it ubuntu /bin/bash`.  
   - Внутри контейнера установлен `curl` через `apt-get install curl`.  
   - Проверено наличие `curl` командой `curl --version`.  
   - Выход из контейнера (`exit`).

3. **Запустила контейнер с nginx (так как образа не было, он скачался). Работает на локальном хосте, всё корректно:**  
   - Выполнена команда: `docker run -d -p 80:80 nginx`.  
   - Проверено, что сервер доступен: `curl http://localhost`.  
   - Просмотр логов контейнера: `docker logs <container_id>`.  
   - Подключение к контейнеру: `docker exec -it <container_id> /bin/bash`.

4. **Посмотрела запущенные контейнеры и все контейнеры. Остановила контейнер, далее запустила остановленный контейнер и снова остановила для его дальнейшего удаления. Удалила образ:**  
   - Список запущенных контейнеров: `docker ps`.  
   - Список всех контейнеров: `docker ps -a`.  
   - Остановка контейнера: `docker stop <container_id>`.  
   - Запуск остановленного контейнера: `docker start <container_id>`.  
   - Повторная остановка: `docker stop <container_id>`.  
   - Удаление контейнера: `docker rm <container_id>`.
