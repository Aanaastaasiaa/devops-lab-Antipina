## Выполненные задания

Установка и подготовка
Установленные компоненты:

Python - базовая среда выполнения

MkDocs - система генерации документации

Тема Material - для стилизации сайта

1. **Установка и подготовка:**  
   - Установленные компоненты: Python, MkDocs.  
   - Создана структура проекта: инициализирован проект командой mkdocs new и сформирована базовая структура папок:  
     docs/ - папка с контентом;  
     mkdocs.yml - конфигурационный файл;  
     index.md - главная страница.  
 

2. **Этап 2: Настройка конфигурации**  
    - Настроены базовые параметры сайта в [mkdocs.yml](https://github.com/Aanaastaasiaa/site-antipina)  
    - Добавлена тема Material с расширенными функциями.  
    - Создана структура навигации.  

3. **Этап 3: Разработка контента**  
     Cозданные страницы:  
      - index.md - главная страница с приветствием;  
      - about.md - информация о себе;  
      - projects.md - описание реализованных проектов;  
      - contacts.md - контактная информация.  

4. **Этап 4: Дополнительная функциональность**  
     Реализованные элементы:  
     - Добавлена папка для изображений;  
     - Настроен поиск по сайту;  
     - Созданы дополнительные разделы;  
     - Добавлены медиафайлы.  

5. **Этап 5: Кастомизация дизайна**  
     Настроенные элементы:  
     - Цветовая схема сайта;  
     - Логотип;  
     - Нижний колонтитул;  
     - Социальные ссылки.  

6. **Этап 6: Тестирование и развертывание**  
     Проведенные проверки:  
     - Локальное тестирование через mkdocs serve;  
     - Проверка всех ссылок и функциональности;  
     - Сборка статических файлов.  
     Развертывание:  
     - Папка проекта загружена на Netlify;  
     - Проект передан в гитхаб;  
     - Выполнена публикация сайта.
