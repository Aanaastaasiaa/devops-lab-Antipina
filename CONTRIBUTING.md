# Правила участия в проекте

1. Все изменения вносятся через Pull Request.
2. Название коммита должно быть кратким и понятным (например, "Add README.md").
3. В Pull Request укажите описание изменений.
4. Перед отправкой изменений выполните `git pull origin develop`, чтобы синхронизировать локальную ветку.
